Выбор

Среди всего множества генераторов мне приглянулись Jekyll, Pelican и JBake. Первый из них явный лидер по популярности, но не очень хотелось связываться с Ruby :)

Pelican написан на Python-е, в качестве шаблонизатора используется Jinja, а контент можно писать используя reStructured и Markdown.

JBake написан на Java, в качестве шаблонизатора можно использовать Jade, Thymeleaf, Freemarker и Groovy, а при помощи AsciiDoc, Markdown и старого доброго HTML есть возможность создавать контент.

Jbake

Связка Groovy и HTML меня заинтересовала и выбор пал на JBake.
